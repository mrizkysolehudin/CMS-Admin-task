## Environment Variables

```bash
REACT_APP_API_URL=
```

## Installation

Clone the project

```bash
  git clone link-to-myproject my-project
```

Go to the project directory

```bash
  cd my-project
```

Add the environment variables to .env file

```bash
  REACT_APP_API_URL=
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
